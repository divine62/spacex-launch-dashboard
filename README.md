# 🚀 SpaceX Launch Dashboard

An interactive data visualization dashboard built using **Plotly Dash** to analyze SpaceX launch data. This project enables users to explore launch success rates, payload correlations, and site-wise performance dynamically.

---

## 📊 Features

* 🔽 **Launch Site Selection**

  * Dropdown to filter data by specific launch sites or view all sites

* 🥧 **Success Rate Visualization**

  * Pie chart showing success vs failure rates

* 📈 **Payload vs Success Analysis**

  * Scatter plot to analyze the relationship between payload mass and launch success

* 🎚 **Interactive Payload Filter**

  * Range slider to filter launches by payload mass

---

## 🛠️ Tech Stack

* **Python**
* **Pandas**

## ▶️ How to Run the Project

1. Clone the repository:

```
git clone https://github.com/yourusername/spacex-launch-dashboard.git
```

2. Navigate to the project folder:

```
cd spacex-launch-dashboard
```

3. Install dependencies:

```
pip install pandas dash plotly
```

4. Run the application:

```
python spacex_dash_app.py
```

5. Open in browser:

```
http://127.0.0.1:8050/
```
